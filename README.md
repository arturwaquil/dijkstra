# Dijkstra

Desenvolvido para a cadeira de Teoria dos Grafos e Análise Combinatória, do 2º semestre de Ciência da Computação da UFRGS.

### Compilar

Para compilar, execute o seguinte comando:

```
$ gcc src/main.c src/grafos.c src/lista.c -o dijkstra.exe
```
